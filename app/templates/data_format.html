Data Format

Labels On Tap needs two things for each application: the application data and the label images. The application data is the truth source. The label images are what the app reads. The app compares the scraped label text against the application data.

One Application

Use this shape when you want to test one application with one or more label images.

my-label-test/
  manifest.csv
  images/
    application-001/
      front-label.png
      back-label.png
      neck-label.png

Many Applications

Use this shape when you want to upload a full test set. Each application gets one row in manifest.csv. Each application can have one image or many images.

my-upload/
  manifest.csv
  images/
    application-001/
      front-label.png
      back-label.png
    application-002/
      front-label.jpg
      back-label.jpg
      side-label.jpg

manifest.csv

The manifest tells the app what the application says. The image paths tell the app which label images belong to each application.

filename,panel_filenames,product_type,brand_name,class_type,alcohol_content,net_contents,imported,country_of_origin
application-001,images/application-001/front-label.png;images/application-001/back-label.png,wine,Example Winery,Sauvignon Blanc,12% BY VOL,750 mL,true,France
application-002,images/application-002/front-label.jpg;images/application-002/back-label.jpg,malt_beverage,Old River Brewing,Ale,5% ALC/VOL,1 Pint,false,

Required Fields

filename

A unique name for the application. This should match the folder name you use under images/.

panel_filenames

One or more label image paths. Separate multiple images with semicolons.

brand_name

The brand name entered on the application.

class_type

The class or type entered on the application.

alcohol_content

The alcohol content entered on the application.

net_contents

The net contents entered on the application.

Optional Fields

fanciful_name

The fanciful name, if one exists.

bottler_producer_name_address

The bottler, producer, importer, or address text.

imported

Use true or false.

country_of_origin

Use this for imported products.

Image Rules

  • Use JPG or PNG files.
  • Keep photos clear and readable.
  • Include every label panel tied to the application.
  • If the government warning is on the back label, include the back label.
  • If net contents, alcohol content, or origin text appears on a side or neck label, include that image too.

Public COLA Demo

This demo uses public TTB COLA application data sourced through COLA Cloud. You can browse 300 real public applications. Each application includes the alcohol label images tied to that application and the actual field values from the application record.

Use the application buttons to move between applications. Use the photo buttons to move between label images inside the selected application.

When you click “Parse This Application,” the app reads the label images, extracts the likely label text, and compares the scraped results against the actual application fields shown below. The left column shows the actual application data. The right column shows what the app scraped from the label images.

The Review Policy controls let you choose how decisions are handled. “Human Review Required” keeps every parsed application in a reviewer workflow with accept and reject actions. “Auto-Route Clear Decisions” lets the app automatically route clear pass/fail results while still sending uncertain cases to review.
